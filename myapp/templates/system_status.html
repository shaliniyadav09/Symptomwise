Troubleshooting

Chatbot Issues Detected

The AI chatbot is experiencing technical difficulties. Here's how to fix it:

  1. Open a terminal/command prompt
  2. Run: ollama serve (keep this running)
  3. In another terminal, run: ollama pull symptomwise:latest
  4. Run: python check_ollama_status.py to verify
Quick Fix Commands

Run these commands to automatically fix common issues:

  • python fix_common_issues.py - Fix all common issues
  • python start_system.py - Start all services
  • python check_ollama_status.py - Check AI status
